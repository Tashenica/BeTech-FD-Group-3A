# BeTech-FD-Group-3A
Group 3A Frontend Development Capstone Project 

## Medium-Inspired Landing Page

A collaborative frontend development capstone project created by **Group 3A** as part of the BeTechified Frontend Development programme.

The goal of this project is to create a responsive, visually appealing landing page inspired by the design and structure of Medium.com, using only **HTML and CSS**.

---

## 📌 Project Overview

This project focuses on recreating the key elements of a modern Medium-inspired landing page while applying fundamental frontend development principles.

The project allows our team to practise:

- Semantic HTML
- CSS styling
- Responsive web design
- Flexbox and Grid layouts
- Website structure and component organisation
- Git and GitHub collaboration
- Team-based development workflows
- UI/UX design and visual consistency

Our goal is to create a professional, responsive landing page that meets the project requirements while demonstrating creativity and attention to detail.

---

## 🎯 Project Objectives

The main objectives of this project are to:

- Build a structured landing page using semantic HTML.
- Apply CSS to create a professional and consistent visual design.
- Use Flexbox and/or CSS Grid for page layouts.
- Create a responsive experience for desktop, tablet, and mobile devices.
- Collaborate effectively using GitHub and Git branches.
- Develop a landing page inspired by Medium.com.
- Apply a consistent visual style across all sections of the website.

---

## 🖥️ Technologies Used

This project is built using:

- HTML5
- CSS3
- Google Fonts
  - Playfair Display
  - Source Sans 3
- Git
- GitHub
- Visual Studio Code

No JavaScript framework or backend technology is required for this project.

## 🧩 Website Structure

The landing page is organised into the following main sections:

### 1. Navigation Bar

The navigation area includes:

- Medium logo
- Our Story
- Membership
- Write
- Sign In
- Get Started call-to-action button

### 2. Hero Section

The main introduction area of the landing page.

This section will contain:

- Main heading
- Supporting description
- Call-to-action
- Supporting visual/illustration

### 3. Why Choose Us / Features

This section highlights the main benefits and features of the platform.

### 4. Testimonials

This section showcases user feedback and experiences.

### 5. Footer

The footer provides additional information and copyright details.
